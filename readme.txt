== Description ==

Are you looking for WordPress Side Tab Plugin or Slide Tab Plugin? Slide Out your desire page with this very simple and user friendly Slick Tab Plugin. The pages can contain forms, widgets, menu options, logins, sign-ups, annoying popoup ads etc. There are a few templates included to get you started easily. 
The settings page appears in the Dashboard -> Settings where you can set all your tab slide options such as page source, size, positioning, autohide, opacity. This plugin is very helpful for easy and user friendly settings. 

Display any content that you wish! Below are some of its cool features and functions:

== Installation ==

1. Upload the `slick-tab` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done!
4. Go to Dhasboard -> Settings -> Slick Tab for more options ans settings.
